## ✨ Features

- 🎨 **Modern UI** — Sleek glassmorphism design with an accessible dark/light mode toggle.
- 📱 **Fully Responsive** — Beautiful layouts adapted perfectly for mobile, tablet, and desktop screens.
- ⚡ **Zero Dependencies** — Built with pure HTML5, Vanilla CSS3, and Vanilla JavaScript (ES6+). No heavy frameworks.
- 🔒 **Privacy First** — 100% of computations happen locally on your browser. Your data is entirely yours. Local storage is used for things like the auto-saving scratchpad.
- ♿ **Accessible** — Built with ARIA roles, semantic markup, and WCAG-compliant color contrasts.
- 🔍 **SEO Optimized** — 39 dynamically routed canonical URLs, automated `sitemap.xml`, Open Graph & Twitter Cards, and JSON-LD structured data.

## 🚀 Getting Started

Utilvault is a fully static site! Zero build steps, zero configuration needed to run.

### Run Locally

```bash
# Clone the repository
git clone https://github.com/rizvee/Utilvault.git

# Enter the directory
cd Utilvault

# Simply open index.html in any modern browser!
# Or, if you prefer using a local server for testing:
npx serve .
```

### Deploy to GitHub Pages

1. Fork or push the repository to GitHub.
2. Go to **Settings → Pages**.
3. Set the source to the `main` branch and root `/` folder.
4. Your site will instantly go live at `https://<username>.github.io/Utilvault/`!
